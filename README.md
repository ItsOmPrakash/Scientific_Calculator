# Scientific_Calculator

This is my first project for the Pinnacle Labs Internship, Where i have built a Scientific Calculator that also includes some 
other features as well such as BMI calculation, discount calculation, and various unit converters. I have used Python and Tkinter for this project.

## Features

Scientific calculator with essential math functions
BMI calculator
Discount calculator
Speed, length, and temperature converters
Desktop GUI application (It runs without Browser using Tkinter)

## How to Run
Make sure that you have installed Python correctly then run: python main.py
